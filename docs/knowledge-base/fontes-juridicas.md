## Hierarquia Normativa (Art. 59, CF/88)

```
1. Constituição Federal
2. Emendas Constitucionais
3. Tratados Internacionais de Direitos Humanos (art. 5°, §3°, CF — status constitucional)
4. Tratados Internacionais comuns (status supralegal — RE 466.343 STF)
5. Leis Complementares
6. Leis Ordinárias
7. Leis Delegadas
8. Medidas Provisórias
9. Decretos Legislativos
10. Resoluções do Congresso
11. Decretos do Executivo
12. Normas infralegais (portarias, instruções normativas, resoluções de órgãos)
13. Legislação estadual (Constituições Estaduais, Leis Estaduais, Decretos Estaduais)
14. Legislação municipal (Leis Orgânicas, Leis Municipais, Decretos Municipais)
```

## Regra de Verificação para Agentes

### Antes de citar qualquer dispositivo, o agente DEVE:

```
1. IDENTIFICAR o nível normativo (federal, estadual, municipal, regulatório)
2. VERIFICAR a vigência atual (não foi revogado? houve alteração recente?)
3. CONFIRMAR a aplicabilidade ao caso (jurisdição, competência, matéria)
4. CHECAR legislação estadual/municipal quando o caso exigir
5. VERIFICAR se há precedente vinculante sobre o tema
6. INDICAR a fonte de consulta na citação
```

### Sinais de alerta (o agente deve sinalizar):

- Lei anterior a 2015 → verificar se não foi revogada ou alterada
- Súmula anterior à Reforma Trabalhista (2017) → verificar aplicabilidade
- Jurisprudência anterior à vigência do CPC/2015 → pode estar superada
- Norma regulatória → verificar se não foi substituída por nova resolução
- Legislação estadual → confirmar que é a do estado correto do processo
- Medida provisória → verificar se foi convertida em lei ou caducou
