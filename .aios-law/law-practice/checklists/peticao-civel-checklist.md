# Checklist: Petição Cível

## Requisitos Formais (CPC)
- [ ] Endereçamento ao juízo competente
- [ ] Qualificação completa do autor (nome, CPF, RG, endereço, profissão)
- [ ] Qualificação completa do réu
- [ ] Fatos narrados cronologicamente
- [ ] Fundamentação jurídica presente
- [ ] Pedido certo e determinado
- [ ] Valor da causa indicado e fundamentado (art. 292, CPC)
- [ ] Provas requeridas
- [ ] Documentos essenciais anexados (art. 320, CPC)
- [ ] Procuração anexa

## Fundamentação
- [ ] Artigos de lei citados e vigentes
- [ ] Jurisprudência com referência completa
- [ ] Causa de pedir coerente com pedidos
- [ ] Sem contradições internas

## Pedidos
- [ ] Pedidos compatíveis entre si
- [ ] Pedido de honorários advocatícios
- [ ] Pedido de justiça gratuita (se aplicável)
- [ ] Pedido de tutela de urgência (se aplicável)

## Conformidade OAB
- [ ] Linguagem respeitosa
- [ ] Sem promessa de resultado
- [ ] Fundamentação presente

## Formatação
- [ ] Fonte adequada (Times New Roman ou Arial)
- [ ] Espaçamento 1,5
- [ ] Parágrafos com recuo
- [ ] Páginas numeradas
- [ ] Data e assinatura
