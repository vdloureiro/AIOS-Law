# Checklist: Contrato

## Estrutura
- [ ] Preâmbulo com qualificação completa das partes
- [ ] Objeto claramente definido
- [ ] Prazo e vigência
- [ ] Preço e forma de pagamento
- [ ] Obrigações de cada parte
- [ ] Cláusula de rescisão
- [ ] Cláusula de foro
- [ ] Espaço para assinaturas
- [ ] Espaço para testemunhas (2)

## Legalidade
- [ ] Sem cláusulas abusivas (art. 51, CDC se aplicável)
- [ ] Sem cláusulas leoninas
- [ ] Objeto lícito
- [ ] Capacidade das partes
- [ ] Forma adequada (instrumento público se exigido por lei)

## Proteção do Cliente
- [ ] Cláusula de confidencialidade
- [ ] Multas proporcionais
- [ ] Direito de rescisão com aviso prévio razoável
- [ ] Limitação de responsabilidade
- [ ] Foro favorável ou neutro

## Clareza
- [ ] Sem ambiguidades
- [ ] Termos definidos
- [ ] Redação clara e objetiva
- [ ] Numeração consistente
