# Checklist: Triagem / Intake

## Dados do Cliente
- [ ] Nome completo coletado
- [ ] CPF/CNPJ coletado
- [ ] Contato (telefone + email)
- [ ] Endereço completo

## Fatos
- [ ] Relato cronológico obtido
- [ ] Datas relevantes identificadas
- [ ] Partes envolvidas identificadas
- [ ] Local dos fatos (jurisdição) identificado
- [ ] Tentativas prévias de resolução verificadas

## Documentos
- [ ] Documentos disponíveis listados
- [ ] Documentos faltantes identificados
- [ ] Cópias recebidas e registradas

## Classificação
- [ ] Área do direito identificada
- [ ] Urgência avaliada
- [ ] Complexidade estimada

## Encaminhamento
- [ ] Ficha do caso criada
- [ ] Encaminhamento definido
- [ ] Case-manager notificado
