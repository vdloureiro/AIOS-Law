# Task: Revisar Conformidade com Código de Ética OAB

## Metadata
- id: revisar-conformidade-oab
- agent: @reviewer
- type: quality-gate
- severity: BLOCK

## Objetivo
Verificar se o documento está em conformidade com o Código de Ética da OAB.

## Checklist

### Proibições (Art. 2° e seguintes do Código de Ética)
- [ ] NÃO promete resultado
- [ ] NÃO faz captação indevida de clientela
- [ ] NÃO deprecia outros advogados
- [ ] NÃO falta com respeito ao Judiciário
- [ ] NÃO usa linguagem ofensiva contra as partes
- [ ] NÃO viola sigilo profissional
- [ ] NÃO faz publicidade irregular

### Deveres (Art. 2° do Código de Ética)
- [ ] Mantém urbanidade e respeito
- [ ] Linguagem técnica e adequada
- [ ] Fundamentação presente (não mera opinião)
- [ ] Independência profissional preservada

### LGPD
- [ ] Dados pessoais tratados conforme necessário
- [ ] Não há exposição desnecessária de dados sensíveis

## Veredicto
- APROVADO: Em total conformidade
- AJUSTES: Pequenas adequações de linguagem
- REPROVADO: Violação identificada — requer reescrita

## Output
- Relatório de conformidade OAB
- Apontamentos específicos
