# Task: Revisar Formatação e Linguagem

## Metadata
- id: revisar-formatacao
- agent: @reviewer
- type: quality-gate
- severity: INFO

## Objetivo
Verificar se a peça segue os padrões de formatação do tribunal destinatário e utiliza linguagem técnica adequada.

## Checklist

### Endereçamento
- [ ] Juízo competente corretamente identificado
- [ ] Tratamento adequado à autoridade (Excelentíssimo, Meritíssimo, etc.)
- [ ] Formato conforme praxe do tribunal (Vara, Seção, Câmara, Turma)
- [ ] Referência ao número do processo (quando aplicável)

### Qualificação das Partes
- [ ] Nome completo
- [ ] Nacionalidade, estado civil, profissão
- [ ] CPF/CNPJ
- [ ] Endereço completo
- [ ] Representação por advogado com OAB

### Formatação Visual
- [ ] Fonte adequada (Times New Roman ou Arial, 12pt)
- [ ] Espaçamento entre linhas de 1,5
- [ ] Margens conforme padrão (3cm superior, 2cm demais)
- [ ] Recuo de parágrafo de 2cm
- [ ] Numeração de páginas
- [ ] Títulos e subtítulos destacados

### Linguagem e Estilo
- [ ] Ortografia e gramática corretas
- [ ] Vocabulário jurídico adequado e preciso
- [ ] Sem coloquialismos ou informalidades
- [ ] Parágrafos bem estruturados (não excessivamente longos)
- [ ] Citações legais em formato padrão
- [ ] Expressões em latim em itálico (quando utilizadas)
- [ ] Siglas definidas no primeiro uso

### Citações e Referências
- [ ] Legislação: Lei n° X.XXX/AAAA, art. XX
- [ ] Jurisprudência: Tribunal, Turma/Câmara, Tipo Recurso n° XXXXX, Relator(a), Data
- [ ] Doutrina: AUTOR, Título, Editora, Ano, p. XX
- [ ] Súmulas: Súmula n° XXX do STF/STJ/TST

### Estrutura da Peça
- [ ] Seções na ordem convencional (Fatos → Direito → Pedidos)
- [ ] Pedidos numerados e claros
- [ ] Valor da causa indicado (quando obrigatório)
- [ ] Data, local e assinatura ao final
- [ ] Documentos anexos listados

### Protocolamento Digital (PJe/e-SAJ/PROJUDI)
- [ ] Formato compatível com o sistema do tribunal
- [ ] Tamanho do arquivo dentro dos limites
- [ ] Documentos anexos nomeados conforme padrão

## Padrões por Tribunal

| Tribunal | Particularidades |
|----------|-----------------|
| Justiça Estadual (TJSP) | e-SAJ, formato PDF, máx. 10MB por arquivo |
| Justiça Federal (TRF3) | PJe, formato PDF, máx. 10MB |
| Justiça do Trabalho (TRT2) | PJe-JT, formato PDF, máx. 8MB |
| Tribunais Superiores | Peticionamento eletrônico próprio |

## Veredicto
- ADEQUADO: Formatação e linguagem dentro dos padrões
- AJUSTES: Pequenas correções de formatação ou linguagem
- INADEQUADO: Formatação ou linguagem fora dos padrões do tribunal

## Output
- Relatório de formatação e linguagem
- Lista de ajustes necessários
- Veredicto
