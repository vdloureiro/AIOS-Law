# Task: Revisar Prazos Processuais

## Metadata
- id: revisar-prazos
- agent: @reviewer | @case-manager
- type: quality-gate
- severity: BLOCK

## Objetivo
Verificar se todos os prazos processuais mencionados ou relevantes estão corretos.

## Checklist

### Tempestividade
- [ ] A peça está sendo apresentada dentro do prazo legal
- [ ] Contagem em dias úteis (CPC art. 219) está correta
- [ ] Feriados e suspensões foram considerados
- [ ] Data de intimação/publicação está correta

### Prazos Mencionados na Peça
- [ ] Prazos prescricionais/decadenciais citados estão corretos
- [ ] Base legal dos prazos está indicada
- [ ] Cálculo dos prazos está correto

### Próximos Prazos
- [ ] Prazo para contra-razões/resposta da parte contrária identificado
- [ ] Prazos futuros cadastrados no sistema

## Output
- Confirmação de tempestividade
- Lista de próximos prazos
