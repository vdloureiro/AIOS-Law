# Task: Coleta de Documentos

## Metadata
- id: coleta-documentos
- agent: @intake
- type: checklist

## Objetivo
Verificar e coletar todos os documentos necessários para o caso.

## Documentos por Área

### Civil Geral
- [ ] Documentos de identificação (RG, CPF)
- [ ] Comprovante de residência
- [ ] Contrato (se aplicável)
- [ ] Notificações extrajudiciais
- [ ] Comprovantes de pagamento
- [ ] Fotos/prints/evidências
- [ ] Declaração de hipossuficiência (se aplicável)

### Trabalhista
- [ ] CTPS (digital ou física)
- [ ] Contracheques (últimos 12 meses)
- [ ] TRCT (Termo de Rescisão)
- [ ] Extrato do FGTS
- [ ] Comprovante de seguro-desemprego
- [ ] Acordo/Convenção coletiva
- [ ] Contrato de trabalho
- [ ] Atestados médicos (se aplicável)
- [ ] PPP - Perfil Profissiográfico (se insalubre/periculoso)

### Criminal
- [ ] Boletim de ocorrência
- [ ] Auto de prisão em flagrante (se aplicável)
- [ ] Decisão/despacho judicial
- [ ] Processo/inquérito (se já existente)
- [ ] Laudo pericial (se existente)

### Empresarial
- [ ] Contrato social / Estatuto
- [ ] Alterações contratuais
- [ ] Atas de reunião/assembleia
- [ ] Balanços e demonstrações financeiras
- [ ] Certidões negativas

## Output
- Checklist de documentos preenchido
- Lista de documentos pendentes
- Status: completo / parcial / insuficiente
