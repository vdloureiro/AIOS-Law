# Task: Buscar Legislação

## Metadata
- id: buscar-legislacao
- agent: @researcher
- type: research

## Objetivo
Pesquisar legislação aplicável ao tema, verificando vigência e alterações.

## Metodologia

### 1. Identificar Diplomas Legais
- [ ] Constituição Federal (artigos aplicáveis)
- [ ] Leis complementares
- [ ] Leis ordinárias
- [ ] Medidas provisórias
- [ ] Decretos regulamentadores
- [ ] Resoluções e normas infralegais

### 2. Verificar Vigência
- [ ] Cada dispositivo está vigente?
- [ ] Houve alteração recente?
- [ ] Há revogação total ou parcial?
- [ ] Há ADI pendente sobre o dispositivo?

### 3. Verificar Regulamentação
- [ ] O dispositivo foi regulamentado?
- [ ] A regulamentação está vigente?

### 4. Documentar
- [ ] Transcrição do dispositivo
- [ ] Diploma legal, número e data
- [ ] Status de vigência
- [ ] Alterações relevantes

## Output
- Relatório de legislação aplicável
- Status de vigência de cada dispositivo
- Histórico de alterações relevantes
