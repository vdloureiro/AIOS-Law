# Task: Buscar Jurisprudência

## Metadata
- id: buscar-jurisprudencia
- agent: @researcher
- type: research

## Objetivo
Pesquisar jurisprudência relevante sobre tema específico, de forma completa e verificável.

## Metodologia

### 1. Definir Escopo
- [ ] Tema central
- [ ] Sub-temas
- [ ] Período de busca (priorizar últimos 2 anos)
- [ ] Tribunais relevantes
- [ ] Palavras-chave

### 2. Pesquisa Hierárquica
- [ ] Súmulas Vinculantes (STF)
- [ ] Temas de Repercussão Geral (STF)
- [ ] Recursos Repetitivos (STJ, art. 1.036, CPC)
- [ ] Súmulas do STJ
- [ ] Súmulas do STF (não vinculantes)
- [ ] OJs do TST (se trabalhista)
- [ ] IRDRs dos TRFs/TJs
- [ ] Acórdãos recentes relevantes

### 3. Classificar Resultados
- [ ] Posição MAJORITÁRIA vs MINORITÁRIA
- [ ] PACIFICADA vs EM DISCUSSÃO
- [ ] Tendência: CONSOLIDANDO vs MUDANDO vs ESTÁVEL

### 4. Documentar
- [ ] Ementa completa de cada julgado relevante
- [ ] Referência completa (tribunal, turma, relator, data, número do processo)
- [ ] Classificação de relevância

## Restrições
- NUNCA inventar jurisprudência
- NUNCA alterar ementas
- NUNCA omitir posição contrária

## Output
- Relatório de pesquisa jurisprudencial
- Organizado por tese/sub-tese
- Com referências completas e verificáveis
