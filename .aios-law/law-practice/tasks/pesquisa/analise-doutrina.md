# Task: Análise Doutrinária

## Metadata
- id: analise-doutrina
- agent: @researcher
- type: research

## Objetivo
Pesquisar e consolidar posições doutrinárias relevantes sobre tema jurídico específico, identificando correntes majoritárias, minoritárias e autores de referência.

## Metodologia

### 1. Definir Escopo
- [ ] Tema central e sub-temas
- [ ] Área do direito
- [ ] Questão jurídica específica a ser respondida

### 2. Identificar Autores de Referência
- [ ] Doutrinadores clássicos da área (obras fundamentais)
- [ ] Doutrinadores contemporâneos (posições atualizadas)
- [ ] Autores especializados no sub-tema
- [ ] Autores que influenciam a jurisprudência dos tribunais superiores

### 3. Mapear Correntes Doutrinárias
- [ ] Corrente majoritária — posição predominante na doutrina
- [ ] Corrente minoritária — posições divergentes relevantes
- [ ] Novas correntes — tendências doutrinárias emergentes
- [ ] Fundamentos de cada corrente

### 4. Correlacionar com Jurisprudência
- [ ] Quais correntes doutrinárias são adotadas pelo STF/STJ
- [ ] Quais autores são mais citados nos acórdãos
- [ ] Há divergência entre doutrina e jurisprudência

### 5. Documentar
- [ ] Referência completa: AUTOR. *Título*. Editora, Edição, Ano, p. XX
- [ ] Transcrição fiel das passagens relevantes
- [ ] Classificação da posição do autor

## Hierarquia de Fontes Doutrinárias

```
1. Tratados e manuais consagrados
2. Comentários a códigos e leis
3. Monografias especializadas
4. Artigos em revistas jurídicas indexadas
5. Teses e dissertações
6. Pareceres publicados
```

## Formato do Relatório

```markdown
# PESQUISA DOUTRINÁRIA

**Tema:** {tema}
**Área:** {area}
**Data:** {data}
**Pesquisador:** Dra. Sophia

## 1. Corrente Majoritária
**Posição:** {descricao}
**Autores:** {lista_autores}
**Fundamento:** {fundamentacao}

## 2. Corrente Minoritária
**Posição:** {descricao}
**Autores:** {lista_autores}
**Fundamento:** {fundamentacao}

## 3. Correlação com Jurisprudência
{analise_correlacao}

## 4. Conclusão
{posicao_recomendada_com_justificativa}

## Referências
{lista_completa_referencias}
```

## Restrições
- NUNCA inventar autores, obras ou citações
- NUNCA atribuir posição a autor sem referência verificável
- NUNCA omitir correntes contrárias relevantes
- SEMPRE indicar edição e ano da obra

## Output
- Relatório de pesquisa doutrinária
- Lista de referências bibliográficas completas
- Mapeamento de correntes com autores
