# Task: Pesquisa de Direito Comparado

## Metadata
- id: direito-comparado
- agent: @researcher
- type: research

## Objetivo
Pesquisar como ordenamentos jurídicos estrangeiros tratam questão jurídica específica, identificando soluções normativas e jurisprudenciais que possam fundamentar ou enriquecer a argumentação.

## Metodologia

### 1. Definir Escopo
- [ ] Questão jurídica a ser comparada
- [ ] Jurisdições relevantes para comparação
- [ ] Objetivo da comparação (fundamentar tese, identificar tendências, buscar soluções)

### 2. Selecionar Jurisdições
Priorizar conforme o tema:

| Tema | Jurisdições Prioritárias |
|------|-------------------------|
| Direito Civil / Contratos | Portugal, Alemanha, França, Itália |
| Direito Penal | Alemanha, Espanha, Portugal, Argentina |
| Direito do Trabalho | Portugal, Espanha, França, OIT |
| Direito Tributário | EUA, Alemanha, OCDE |
| Direito Empresarial | EUA (Delaware), Reino Unido, Alemanha |
| Direito Digital / LGPD | UE (GDPR), EUA (CCPA), Argentina |
| Direito Constitucional | Alemanha, EUA, Portugal, Colômbia |
| Direito Administrativo | França, Portugal, Espanha |

### 3. Pesquisar por Jurisdição
- [ ] Legislação aplicável (texto legal, artigos específicos)
- [ ] Jurisprudência dos tribunais superiores
- [ ] Doutrina de referência
- [ ] Tratados e convenções internacionais aplicáveis

### 4. Analisar Comparativamente
- [ ] Semelhanças com o direito brasileiro
- [ ] Diferenças relevantes
- [ ] Soluções inovadoras aplicáveis ao caso
- [ ] Tendências internacionais

### 5. Avaliar Aplicabilidade
- [ ] Compatibilidade com o ordenamento brasileiro
- [ ] Precedentes do STF/STJ que utilizaram direito comparado
- [ ] Limitações da transposição

## 3. Aplicabilidade ao Caso Brasileiro
{analise_aplicabilidade}

## Restrições
- NUNCA inventar legislação ou jurisprudência estrangeira
- NUNCA apresentar solução estrangeira como diretamente aplicável sem análise de compatibilidade
- SEMPRE indicar as diferenças de sistema jurídico (civil law vs common law)
- SEMPRE citar fontes verificáveis

## Output
- Relatório de direito comparado
- Quadro comparativo entre jurisdições
- Análise de aplicabilidade ao caso brasileiro
