# Task: Análise de Viabilidade Jurídica

## Metadata
- id: analise-viabilidade
- agent: @strategist
- type: analysis

## Objetivo
Analisar a viabilidade jurídica do caso, identificar riscos, mapear teses e recomendar estratégia.

## Pré-condições
- [ ] Ficha do caso disponível
- [ ] Fatos coletados pela @intake
- [ ] Documentos relevantes disponíveis

## Etapas

### 1. Análise dos Fatos
- [ ] Revisar cronologia dos fatos
- [ ] Identificar fatos jurídicamente relevantes
- [ ] Verificar consistência do relato

### 2. Questão Jurídica
- [ ] Identificar questão(ões) jurídica(s) central(is)
- [ ] Listar sub-questões derivadas

### 3. Fundamentação Legal
- [ ] Identificar legislação aplicável
- [ ] Solicitar pesquisa jurisprudencial a @researcher (se necessário)
- [ ] Consultar doutrina relevante

### 4. Teses Jurídicas
- [ ] Mapear tese principal com fundamentação
- [ ] Identificar teses alternativas
- [ ] Antecipar contra-argumentos da parte contrária

### 5. Análise de Riscos
- [ ] Riscos jurídicos (tese fraca, jurisprudência contrária)
- [ ] Riscos processuais (prescrição, decadência, competência)
- [ ] Riscos financeiros (custas, sucumbência, honorários periciais)

### 6. Probabilidade de Êxito
- [ ] Classificar: Alta (>70%) | Moderada (40-70%) | Baixa (15-40%) | Temerária (<15%)
- [ ] Justificar a classificação

### 7. Recomendação Estratégica
- [ ] Via recomendada (judicial/extrajudicial)
- [ ] Tipo de ação/procedimento
- [ ] Competência (Justiça/Vara)
- [ ] Pedidos recomendados
- [ ] Provas a produzir
- [ ] Considerar vias alternativas (mediação, arbitragem, acordo)

### 8. Custo-Benefício
- [ ] Custas processuais estimadas
- [ ] Tempo estimado de tramitação
- [ ] Valor da causa vs custos totais

## Output
- Parecer de viabilidade (template: parecer-juridico-tmpl.md)
- Classificação de probabilidade
- Recomendação estratégica
- Mapeamento de riscos
